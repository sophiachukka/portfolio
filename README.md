# Portfolio

Personal portfolio website ready to deploy with Vercel.

## Customize

Edit `index.html` and replace:

- `Your Name`
- `hello@example.com`
- project titles and descriptions
- LinkedIn, GitHub, and resume links

## Deploy With Vercel

1. Create a GitHub repository named `portfolio`.
2. Upload or push these files to that repository.
3. Go to [Vercel](https://vercel.com/new).
4. Import the `portfolio` GitHub repository.
5. Keep the default settings and deploy.

Because this is a static website, Vercel does not need a build command.
